# Interview Workspace

`interview/` 目录用于维护可复用的项目介绍、技术 topic bank，以及面向具体公司的定制面试包。

## 通用文件

- `interview.md`：通用项目介绍和技术面试素材
- `coding/`：coding 题和实现草稿

## 公司目录建议

每个公司或岗位可以单独建一个子目录，例如：

- `interview/binance/`
- `interview/tiktok-data-science/`

每个目录通常包含：

- `jd.md`
- `mock.md`
- `my-q.md`

推荐顺序：

1. 先放入 `jd.md`
2. 再基于 JD 和现有 lib 生成 `mock.md`
3. 最后整理 `my-q.md`
