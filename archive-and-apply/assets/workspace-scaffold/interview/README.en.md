# Interview Workspace

The `interview/` folder stores reusable project introductions, topic-bank notes, and company-specific interview prep packs.

## Shared Files

- `interview.md`: reusable project intros and technical talking points
- `coding/`: coding questions and implementation drafts

## Suggested Company Folders

Each target company or role can have its own subfolder, for example:

- `interview/binance/`
- `interview/tiktok-data-science/`

Each company folder usually contains:

- `jd.md`
- `mock.md`
- `my-q.md`

Recommended order:

1. Save `jd.md` first
2. Generate `mock.md` from the JD and the existing workspace
3. Refine `my-q.md` last
