# Statement of Purpose (SOP) / 个人陈述

> 用途：从 source entries 生成申请用的个人陈述初稿，通常 800-1000 词，每所学校需要定制。
> 状态：模板

## 使用原则

- SOP 是叙事，不是简历罗列。要写"我为什么想做这个"和"我做了什么让我适合做这个"。
- 先从 source entries 提取事实，再写成连贯的故事线。
- 每所学校定制：换学校名、研究组名、导师名、课程名；核心叙事可以复用。
- 长度通常 800-1000 词（美国），500-800 词（英国/欧亚），部分项目允许更短。
- 不要编造成绩或经历， claims 必须可回溯到 source entries。

## 推荐结构

```md
# Statement of Purpose

## 第一段：开场钩子 (Hook)
用 1-2 句引人注目的话开场，引出你的研究或职业兴趣。

## 第二段：研究 / 项目经历 (Body 1)
- 你做了什么（具体问题 + 技术方案 + 结果）
- 为什么重要
- 从中学到了什么（方法论 / 技术 / 视野）

## 第三段：继续深化 (Body 2)
- 如果有第二段研究经历，写在这里
- 或写一段课程/自学经历，展示你的技术积累

## 第四段：为什么选这个项目 (Why this program)
- 学校的哪个实验室 / 教授 / 课程吸引你
- 你能给这个项目带来什么

## 第五段：未来方向 (Future goals)
- 读博 / 职业方向
- 长期愿景

## 第六段：结语 (Closing)
- 简短总结 + 期待
```

## 按项目类型定制

### PhD / 研究型硕士

- Research focus 占 60-70%
- 必须点名潜在导师或实验室
- 展示独立研究能力，不只是上课

### MBA / 职业型硕士

- Leadership + impact 占 40-50%
- 叙事弧：从挑战 → 行动 → 结果
- 展示 career progression 逻辑

### 混合方向 (MPP / MEd / MFA)

- 政策分析 / 教育 / 艺术叙事各有不同重心
- 参见对应 section
